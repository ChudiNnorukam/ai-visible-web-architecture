# Support

## What This Repository Supports

This repository supports:

- questions about the architecture pattern documented here
- corrections to verified public facts about `chudi.dev`
- improvements to diagrams, examples, and verification scripts

## Best Channel

- Use GitHub Issues for documentation gaps, broken links, or reproducible repo problems.
- Use pull requests for specific fixes with evidence.
- Use `hello@chudi.dev` for security reports or sensitive concerns.

## What To Include

When asking for help, include:

- the page, doc, or script involved
- what you expected
- what you observed instead
- the command you ran, if the issue is script-related
