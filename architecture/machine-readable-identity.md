# Machine-Readable Identity

## Definition

Machine-readable identity is the layer that lets search engines and AI systems understand who a site belongs to, what topics it covers, and where its authority is concentrated.

## Components

- canonical site identity
- consistent author profile data
- structured data for entities and collections
- topic-level organization instead of flat archive sprawl
- linked discovery files for machine access

## Applied to chudi.dev

`chudi.dev` uses one coherent identity graph:

- a single author entity
- explicit topic clusters
- structured metadata across homepage, about page, and posts
- discovery files that reinforce where authority lives

The result is that the site is easier to treat as a knowledge node rather than a disconnected set of posts.
